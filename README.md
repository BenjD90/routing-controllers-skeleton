# n9-node-routing
## Easily create epress app in TypeScript with Decorators 
## Wrapper of project [routing-controllers](https://github.com/typestack/routing-controllers)

1. Install all dependencies:

    `yarn`
    
2. Setup git hooks

   `yarn run init`

3. Run the project tests:

    `yarn test`
